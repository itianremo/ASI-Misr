Getting Help

This procedure enables you to display the WebHelp file associated with the ASI-CRM application.

To activate the ASI-CRM WebHelp, perform the following steps:
  1. On the upper right corner of your active page, click the Help button.
  2. The WebHelp file is activated in a two-pane mode. The left shows the Help contents; the right shows the active topic.
