Browsing the ASI-CRM Application

As ASI-CRM is a Web-based application, all browsing and navigation features are available to take you from one page to another with ease and flexibility.

What do you want to do?
  • View the main menu options
  • View the toolbar options
  • Browse customer account or contact records
  • View the data sorting options
  • View the tagging and data filtering options
  • Activate the application Help

 

View the main menu options

The following is a description of the options available in the main menu of the ASI-CRM application:

Option Description

Opens the Accounts page where you can

  • Search for a company or a business sector to view its saved data
  • Tag or un-tag one or all account records

Opens the Account Details page where you can

  • Add a new customer account
  • Search for an existing customer account
  • Edit an exiting customer account

Opens the Contacts page where you can

  • Search for a contact to view its saved data
  • Tag or un-tag one or all contact records

Opens the Contact Details page where you can

  • Add a new customer contact
  • Search for an existing customer contact
  • Edit an existing customer contact

 

Note: Your ASI CRM level of security settings control the pages you can view and use; they also control the operations and/or processes you can perform. Contact your network administrator for detailed information on your ASI CRM level of security settings.

 

View the toolbar options 

The following is a description of the options available in the toolbar of the ASI-CRM application:

Option Description

Saves a transaction or an operation performed on a page or a data record.

Displays the report(s) associated with the displayed page (to be available in later versions).
  • Adds a new customer while you are in the Account Details page.
  • Adds a new contact while you are in the Contact Details page.
  • Edits a customer account record while you are in the Account Details page.
  • Edits a customer contact record while you are in the Contact Details page.
Cancels a transaction or an operation performed on a page or a data record.

 

Browse customer account or contact records

The following is a description of the options that enable you to navigate data records in the ASI-CRM application: 

Option Description

Moves from a displayed record to a next one in the database, based on your search and data sorting criteria.

Moves from a displayed data record to a previous one in the database, based on your search and data sorting criteria.
1,2,3,4, etc.

Moves from one data record page to another.
First Displays the first records available of a database element, accounts or contacts, based on your search and data sorting criteria.
Last Displays the last records available of a database element, accounts or contacts, based on your search and data sorting criteria.
Previous Moves from a displayed record page to a previous one, based on your search and data sorting criteria.
Next Moves from a displayed record page to a next one, based on your search and data sorting criteria.

 

View the data sorting options

The following is a description of the data sorting options available in the Accounts and the Contacts pages that enable you to sort the displayed data records: 

Accounts Page
Option How to use
Tag

Click the Tag column header to sort the displayed data according to its tagging or un-tagging status.
Company/Agency

Click the Company/Agency header to sort the displayed data according to the company or agency name.

  • Click once to perform an A-Z sorting
  • Click for a second time to perform a Z-A sorting.
City

Click the City column header to sort the displayed data according to the city name.

  • Click once to perform an A-Z sorting
  • Click for a second time to perform a Z-A sorting.
State

Click the State column header to sort the displayed data according to the state name.

  • Click once to perform an A-Z sorting
  • Click for a second time to perform a Z-A sorting.

 

Contacts Page
Option Description
Tag Click the Tag column header to sort the displayed data according to its tagging or un-tagging status.
Company/Agent

Click the Company/Agent column header to sort the displayed data according to the company or agent name.

  • Click once to perform an A-Z sorting
  • Click for a second time to perform a Z-A sorting.
First Name

Click the First Name column header to sort the displayed data according to the first name of the contact.

  • Click once to perform an A-Z sorting
  • Click for a second time to perform a Z-A sorting.
Last Name

Click the Last Name column header to sort the displayed data according to the first name of the contact.

  • Click once to perform an A-Z sorting
  • Click for a second time to perform a Z-A sorting.
Telephone

Click the Telephone column header to sort the displayed data according to the contact telephone numbers.
Title

Click the Title column header to sort the displayed data according to the contact salutation titles.
Fax Click the Fax column header to sort the displayed data according to the contact fax numbers.
Email Click the Email column header to sort the displayed data according to the contact email addresses.

View the tagging and data filtering options 

The following is a description of the tagging and data filtering options available in the Accounts and the Contacts pages that enable you to tag and/or filter the displayed data records: 

Tagging
Option Description
Tag All Select Tag All if you want to tag all records displayed on the active Accounts or Contacts page.
Un-Tag All Select Un-Tag All if you want to remove the tagging from all records displayed on the active Accounts or Contacts page.
Filtering
Option Description
All

Select All if you want to display all data records, tagged and un-tagged.
Un-Tagged

Select Un-Tagged if you want to display the un-tagged data records only.
Tagged

Select Tagged if you want to display the tagged data records only.
