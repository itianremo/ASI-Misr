Adding / Editing Customer Notes

This procedure enables you to add/edit notes on ASI's customers.  A note may be used in entering and saving very useful information in relation to a given customer. Later, notes may be used by you or your division manager(s) in taking business decisions concerning that customer.

Prerequisites 

  • You are a member of the division(s); e.g., Customer Relations, that can access the ASI-CRM application.
  • Your CRM security settings allow you to add/edit customer notes.

Note: Contact your network administrator for detailed information on your ASI CRM level of security settings.
To add new customer note(s), perform the following steps:
  1. Log into the ASI-CRM application.
  2. On the main menu bar, click . The Account Details page is opened.
  3. Search for the target customer account you want to add/update notes for. If you want to add a new note, go to the next step; whereas if you want to edit an existing note, select it first, then go to the next step.
  4. Click the Notes tab. The Note Entry and Edit dialog box is displayed for you. Click to see figure.

  5. Place the cursor in the empty editing box and type in your new text. The other fields available in the upper part of this dialog box; e.g., Entered, Edit, Time, and By, are view-only controls and are used to display your information once the note is saved and re-opened.

    Note: In case you want to cancel editing a note and add a new one for the same customer while you are still in this dialog box, click the Add New Note button.

  6. Repeat the above steps to add as many new notes or edit as many existing ones as you want.
  7. Click Save Note , if you want to save the new/updated note to the current customer record.
  8. Click Cancel , if you want to cancel all changes.
